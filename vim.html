Commands:



Navigation:


  • h - Move left.
  • j - Move down.
  • k - Move up.
  • l - Move right.

  • $ - Move to the end of a line.
  • 0 (zero) - Move to the beginning of the current line.
  • gg - Navigate to the first line of the file.
  • G - Navigate to the last line in the file.
  • "number" G - Navigate to a specific line number in the file.
  • "number" "enter" - Go down a "number" of lines from the current position.
  • control + d - Page down in the file.
  • control + b - Page up in the file.


Copy and Paste:


  • yy - Yanks (copies) the current line.
  • "number" yy - Yanks (copies) the specified number of lines.
  • dd - Deletes the current line.
  • "number" dd - Deletes the specified number of lines.
  • dw - Deletes the word at the current position.
  • d$ - Deltes from the cursor's position to the end of the current line.


Working with files:


Opening files with as tabs using the the -p flag.

  • "number"gt - Navigates forward a "number" of tabs.
  • "number"Gt - Navigates backwards a "number" of tabs.
  • :tabnew "filename" - Opens a new file in another tab.
  • :tabclose - Closes the current file.

Opening files with split screen.

  • Open files with the "-o" flag (lowercase letter o) for horizontal split.
  • Open files with the "-O" flag (uppercase letter o) for vertical split.
  • control + "ww" - Navigate between files.
  • :split "filename" - Open up a file with horizontal split.
  • :vs "filename" - Open up a file with vertical split.

Saving and quiting files.

  • :w - Saves the current file you're editing.
  • :wa or :x - Saves all the files you currently have open.
  • :q - Quits the file you're currently editing.
  • :qa - Quits all the files you currently have open.
  • :wqa or :xa - Saves and closes all the files you currently have open.

Working with fils in the forground/background.

  • jobs - from the command line tells you what files are currently in the background.
  • control + "z" - Pushes all files in your current editor session into the background.
  • fg - From the command line bring the editor session currently in the background to the foreground.
  • When working with fg use the ":wa" command to save all files.


Searching and Replacing:


  • :/word - searches forward for the next occurance of "word" in the file.
  • :?word - searches backwards for the next occurance of "word" in the file.
  • :noh - Unhighlights the word/phrase after being searched for.
  • :s/to_replace/replace_with - Replaces the next occurance of the word to replace with the phrase to replace it with.
  • :%s/to_replace/replace_with/gc - Replaces all occurances of the word to replace with the phrase to replace it with. The "g" and "c" at the end are flags for the search and replace which stand for "replace all" and "confirm" before replacing respectively.
